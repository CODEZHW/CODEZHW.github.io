题目： 面试题06. 从尾到头打印链表

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

示例 1：

输入：head = [1,3,2]
输出：[2,3,1]

解题思路：

  • 用递归
  • 用辅助栈

代码：

//递归
ArrayList <Integer> list = new ArrayList <Integer> ( );
    public int[] reversePrint(ListNode head) {
        digui(head);
        int [] res = new int[list.size()];
        for(int i =0; i<list.size(); i++){
            res[i] = list.get(i);
        }
        return res;
    }
    public void digui(ListNode head){
        if(head==null){
            return;
        }
        digui(head.next);
        list.add(head.val);
    }


//辅助栈
public int[] reversePrint(ListNode head) {
        LinkedList<Integer> stack = new LinkedList<Integer>();
        for(ListNode node = head; node!=null; node = node.next){
            stack.add(node.val);
        }
        int [] res = new int[stack.size()];
        int len = stack.size ();
        for(int i =0; i<len; i++){
            res[i] = stack.removeLast();
        }
    return res;
    }

题目：20. 有效的括号

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:

输入: “()”
输出: true
示例 2:

输入: “()[]{}”
输出: true
示例 3:

输入: “(]”
输出: false
示例 4:

输入: “([)]”
输出: false
示例 5:

输入: “{[]}”
输出: true

解题思路：

  • 利用栈先进后出的特点
  • 先将有效括号的左括号作为key 右括号作为value 存入HashMap中
  • 去遍历给到的字符串，如果是有效左括号就将括号入栈
  • 如果不是左括号，弹出栈顶元素，判断是否能与当前括号组成有小括号对，如果可以继续，如果不行返回false.
  • 最后返回栈是否为空（为空说明括号全部匹配，就都是有效括号对）。

代码：

public boolean isValid(String s) {
       if (s.length ( ) % 2 != 0) {
            return false;
        }
        Map <Character, Character> map = new HashMap <> ( );
        map.put ( '(', ')' );
        map.put ( '{', '}' );
        map.put ( '[', ']' );

        Stack <Character> stack = new Stack <> ( );
        for (char c : s.toCharArray ( )) {
            if (map.containsKey ( c )) {
                stack.push ( c );
                continue;
            }
            if (!stack.empty()&&(map.get ( stack.pop ( ) )).equals ( c )) {
                continue;
            }
            return  false;
        }

        return stack.empty ( );
    }

题目： 面试题10- I. 斐波那契数列

解题思路：

  • 递归解决会超时，所以最优解可以使用动态规划。
  • 因为题目的特殊性，可以将空间复杂度压缩至o(1)。

代码：

public int fib(int n) {
        int a = 0, b = 1, sum;
        for(int i = 0; i < n; i++){
            sum = (a + b) % 1000000007;
            a = b;
            b = sum;
        }
        return a;
    }

题目：面试题57 - II. 和为s的连续正数序列

输入一个正整数 target ，输出所有和为 target 的连续正整数序列（至少含有两个数）。

序列内的数字由小到大排列，不同序列按照首个数字从小到大排列。

示例 1：

输入：target = 9
输出：[[2,3,4],[4,5]]

示例 2：

输入：target = 15
输出：[[1,2,3,4,5],[4,5,6],[7,8]]

限制：

  • 1 <= target <= 10^5

解题思路：

  • 滑动窗口的形式

代码：

public int[][] findContinuousSequence(int target) {
    int i = 1; 
    int j = 1; 
    int sum = 0; 
    List<int[]> res = new ArrayList<>();

    while (i <= target / 2) {
        if (sum < target) {
            sum += j;
            j++;
        } else if (sum > target) {
            sum -= i;
            i++;
        } else {
            int[] arr = new int[j-i];
            for (int k = i; k < j; k++) {
                arr[k-i] = k;
            }
            res.add(arr);
            sum -= i;
            i++;
        }
    }
    return res.toArray(new int[res.size()][]);
    }

题目：面试题18. 删除链表的节点

解题思路：

  • 基本操作

代码：

public ListNode deleteNode(ListNode head, int val) {
        if(head ==null){
            return null;
        }
        if(head.val == val){
            return head.next;
        }

        for(ListNode node = head; node.next!=null; node=node.next){
            if(node.next.val == val){
                if(node.next.next==null){
                    node.next = null;
                    return head;
                }
                ListNode next = node.next.next;
                node.next = next;
            }
        }
        return head;
    }

题目：面试题09. 用两个栈实现队列

解题思路：

  • 因为栈与队列结构的区别，栈的栈顶是是队列的队尾，栈的栈尾是队列的队首。
  • 所以当队列进行元素移除的时候，在栈结构中是无法操作的，我们可以把栈反转-即把当前栈的元素转入宁一个栈中，然后执行栈的删除操作。栈1： 3-1-2 => 栈2： 2-1-3，这样可以达到一个队列的效果。
  • 利用LinkedList 代替stack，因为stack继承自Vector,Vector效率很低。

代码：

class CQueue {
    LinkedList<Integer> stack1;
     LinkedList<Integer> stack2;

    public CQueue() {
        stack1 = new LinkedList<>();
        stack2 = new LinkedList<>();
    }

    public void appendTail(int value) {
        stack1.add(value);
    }

    public int deleteHead() {
        if (stack2.isEmpty()) {
            if (stack1.isEmpty()) return -1;
            while (!stack1.isEmpty()) {
                stack2.add(stack1.pop());
            }
            return stack2.pop();
        } else return stack2.pop();
    }
}

题目：面试题42. 连续子数组的最大和

输入一个整型数组，数组里有正数也有负数。数组中的一个或连续多个整数组成一个子数组。求所有子数组的和的最大值。

要求时间复杂度为O(n)。

示例1:

输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。

解题思路：

  • 动态规划

  • 可以用DP[i]表示以元素nums[i]为结尾的最大连续子数组的合。

  • 转义方程：当dp[i-1]<0时，说明dp[i-1] + nums[i] 不如nums[i]本身大

    • 当 dp[i−1]>0 时：执行 dp[i] = dp[i-1] + nums[i]；

    • 当 dp[i−1]≤0 时：执行 dp[i] = nums[i]；

  • 由于 dp[i] 只与 dp[i-1] 和 nums[i] 有关系，因此可以将原数组 nums 用作 dp 列表，即直接在 nums 上修改即可。

  • 由于省去 dp 列表使用的额外空间，因此空间复杂度从 O(N) 降至 O(1) 。

​

代码：

public static int maxSubArray(int[] nums) {
    int res = nums[0];
    for (int i = 1; i < nums.length; i++) {
        nums[i] += Math.max ( nums[i-1], 0 );
        res = Math.max ( res, nums[i] );
    }
    return res;
}

题目：面试题27. 二叉树的镜像

请完成一个函数，输入一个二叉树，该函数输出它的镜像。

例如输入：

 4

/
2 7
/ \ /
1 3 6 9
镜像输出：

 4

/
7 2
/ \ /
9 6 3 1

示例 1：

输入：root = [4,2,7,1,3,6,9]
输出：[4,7,2,9,6,3,1]

解题思路：

  • 递归交换左右节点
  • 可以利用辅助栈

代码：

public TreeNode mirrorTree(TreeNode root) {
        switchroot(root);

        return root;
    }

    public void switchroot(TreeNode root){
        if (root == null){
            return ;
        }
        switchroot(root.left);
        switchroot(root.right);
        TreeNode tmp = root.left;
        root.left = root.right;
        root.right = tmp;
    }

题目：21. 合并两个有序链表

将两个升序链表合并为一个新的升序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

示例：

输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4

解题思路：

  • 递归

代码：

public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
        if (l1 == null) {
            return l2;
        }
        else if (l2 == null) {
            return l1;
        }
        else if (l1.val < l2.val) {
            l1.next = mergeTwoLists(l1.next, l2);
            return l1;
        }
        else {
            l2.next = mergeTwoLists(l1, l2.next);
            return l2;
        }
    }

题目：102. 二叉树的层序遍历

解题思路：

  • BFS
  • 维护一个ArrayList list用于存储树的每层结构，list里面的存储的也是Arraylist即为每层的元素。
  • 递归函数（node-节点，level-当前层次）。
  • 判断当list.size() == level相等的时候，就向list中添加一个数组。
  • 将当前节点元素添加进去，然后递归遍历左右节点，level并加1。

代码：

ArrayList<List<Integer>> list = new ArrayList<List<Integer>> ();

    public List<List<Integer>> levelOrder(TreeNode root) {
        if(root == null){
            return list;
        }
        int level = 0;
        helper(root, level);
        return list;
    }
    public void helper(TreeNode node , int level ){
        if(list.size() == level){
            list.add(new ArrayList<Integer>());
        }
        list.get(level).add(node.val);
        if(node.left!=null){
            helper(node.left, level+1);
        }
        if(node.right!=null){
            helper(node.right, level+1);
        }

    }

题目：104. 二叉树的最大深度

解题思路：

  • 递归遍历，当遍历到的节点的左右节点为null，说明已经遍历到叶子节点返回1。
  • 如果左右节点不为null，递归的去遍历左右节点。
  • 返回 max(左右节点的最大值) +1 ;+1为加上最后的根节点。

代码：

public int maxDepth(TreeNode root) {
        if(root == null){
            return 0;
        }

        if(root.right == null && root.left == null){
            return 1;
        }
        int maxleft  = maxDepth(root.left);
        int maxright = maxDepth(root.right);

        return Math.max(maxleft, maxright) + 1;
    }

题目：111. 二叉树的最小深度

解题思路：

  • 和最大深度差不多的思路
  • 区别在于如果当前某个节点的左/右为空时，说明这个节点就是最路径节点了，可以直接返回

代码：

public int minDepth(TreeNode root) {
        if(root == null){
            return 0;
        }
        if(root.left == null && root.right == null){
            return 1;
        }

        int leftmin = minDepth(root.left);
        int rightmin = minDepth(root.right);

        if(root.left == null || root.right == null) {
            return leftmin + rightmin + 1;
        }

        return Math.min(leftmin, rightmin)+1;
    }

题目：3. 无重复字符的最长子串

给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

示例 1:

输入: “abcabcbb”
输出: 3
解释: 因为无重复字符的最长子串是 “abc”，所以其长度为 3。
示例 2:

输入: “bbbbb”
输出: 1
解释: 因为无重复字符的最长子串是 “b”，所以其长度为 1。
示例 3:

输入: “pwwkew”
输出: 3
解释: 因为无重复字符的最长子串是 “wke”，所以其长度为 3。
请注意，你的答案必须是 子串 的长度，”pwke” 是一个子序列，不是子串。

解题思路：

  • 双指针

代码：

public int lengthOfLongestSubstring(String s) {
        int ans = 0;
        HashMap<Character, Integer> map =  new HashMap<Character, Integer>();
        for(int i=0,j=0; j<s.length();j++){
            char c = s.charAt(j);
            if(map.containsKey(c)){
                i = Math.max(map.get(c), i);
            }
            ans = Math.max(ans , j-i+1);
            map.put (s.charAt(j), j+1 );
        }
        return ans;
    }
