LinkedList源码分析

前言

最近在做算法题，遇到太多需要使用了队列或者栈的情况了，通常情况下我们会使用Java集合包下的Stack。

但是因为Stack是继承自Vector的（线程安全的List集合）所以在性能上会有些许差距。

这个时候就是我们的LinkedList出场了，LinkedList是一个双端链表，可以基于它实现栈或者队列这样的数据结构。(如果你想实现更高效的队列可以使用ArrayDeque)

这里我们主要介绍LinkedList中的一些常用的API。

基本属性

public class LinkedList<E>extends AbstractSequentialList<E>
    implements List<E>, Deque<E>, Cloneable, java.io.Serializable


transient int size = 0;

transient Node<E> first;

transient Node<E> last;

private static class Node<E> {
        E item;
        Node<E> next;
        Node<E> prev;

        Node(Node<E> prev, E element, Node<E> next) {
            this.item = element;
            this.next = next;
            this.prev = prev;
        }
    }

首先可以看到LinkedList继承自AbstractSequentialList，这个类继承自AbstractList。

实现了List、Deque接口；Cloneable接口可以实现克隆，Serializable序列化。

实现栈和队列的关键字段： Node<E> first、 Node<E> last。

还有一个数据节点：Node。它维护了一个双向链表。

//默认的空参构造函数
public LinkedList() {
}

//传入一个集合，它会把集合里面的所有元素添加到链表中去
public LinkedList(Collection<? extends E> c) {
    this();
    addAll(c);
}

addFirst()

//将指定的元素插入链表的头部
public void addFirst(E e) {
    linkFirst(e);
}

private void linkFirst(E e) {
        //获取头节点元素
        final Node<E> f = first;
        //创建一个新的节点，将它的前一个节点置空，下一个节点指向f
        final Node<E> newNode = new Node<>(null, e, f);
        first = newNode;
        //判断头节点是否为空
        if (f == null)
            //说明链表为空，把最后一个节点指针指向新的节点
            last = newNode;
        else
            //不为空，就把原来的头节点的前指针指向新的节点
            f.prev = newNode;
        size++;
        modCount++;
    }

addLast()

//将指定元素追加到链表的尾部
public void addLast(E e) {
    linkLast(e);
}

void linkLast(E e) {
        //获取尾节点的元素
        final Node<E> l = last;
        //创建一个新的节点，它的前一个节点是尾节点，下一个节点是null
        final Node<E> newNode = new Node<>(l, e, null);
        last = newNode;
        //判断为节点是否为null
        if (l == null)
            //说明链表为空，将第一个指针指向新的节点
            first = newNode;
        else
            //不为空，就把原来的尾节点的下一个节点赋值为新建的节点
            l.next = newNode;
        size++;
        modCount++;
    }

removeFirst()

//移除当前链表的第一个元素并返回值(头节点不为空)
public E removeFirst() {
    final Node<E> f = first;
    if (f == null)
        throw new NoSuchElementException();
    return unlinkFirst(f);
}


private E unlinkFirst(Node<E> f) {
        // assert f == first && f != null;
        //获取头节点的值
        final E element = f.item;
        //获取头节点的下一个节点
        final Node<E> next = f.next;
        f.item = null;
        f.next = null; // help GC（主动的将对象引用设为null，帮助垃圾回收）
        //将头节点置为下一个节点
        first = next;
        //如果下一个节点为null
        if (next == null)
            //尾节点置空,因为在链表只有一个元素的时候，头指针和尾指针指向同一个元素
            last = null;
        else
            //新的头节点的前一个节点置空
            next.prev = null;
        size--;
        modCount++;
        return element;
}

removeLast()

//移除链表中的最后一个元素并返回值(尾节点不为空)
public E removeLast() {
    final Node<E> l = last;
    if (l == null)
        throw new NoSuchElementException();
    return unlinkLast(l);
}

private E unlinkLast(Node<E> l) {
        // assert l == last && l != null;
        //获取尾节点
        final E element = l.item;
        //获取尾节点的下一个节点
        final Node<E> prev = l.prev;
        l.item = null;
        l.prev = null; // help GC
        //把尾节点指向前一个节点
        last = prev;
        //如果前一个节点为空
        if (prev == null)
            //头节点置空,因为在链表只有一个元素的时候，头指针和尾指针指向同一个元素
            first = null;
        else
            //前一个节点的下一个节点置空
            prev.next = null;
        size--;
        modCount++;
        return element;
}

peek（）

//返回当前链表的第一个元素
public E peekFirst() {
    final Node<E> f = first;
    return (f == null) ? null : f.item;
 }

//返回当前链表的最后一个元素
public E peekLast() {
        final Node<E> l = last;
        return (l == null) ? null : l.item;
    }

pop()&push()

public E pop() {
    return removeFirst();
}

public void push(E e) {
    addFirst(e);
 }

可以看到我们的POP()和PUSH（）方法分别调用的是removeFirst（）和addFirst（）方法。

可以基于此实现 FIFO的队列结构或者FILO的栈结构。
